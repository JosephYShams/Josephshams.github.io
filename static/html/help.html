Rules

  • You must register all 8 smart home devices.
  • To register a device, you must first answer a configuration test. This configuration test evaluates your understanding of smart home device capabilities and what configuration behaviours reduce privacy risks associated with device capabilities.
  • To help you answer the questions, a description of the device capabilties can be found by clicking the Tips button
  • There is no time limit for this game, although we expect it to last between 15 and 30 minutes.
