Tips

Each smart home device can perform at least three of the following five capabilities.

  • Connectedness - for smart features to be function a solid internet connection is required.
  • Delegation (Automation) - a feature that attempts to perform features on behalf of users.
  • Sensing (Learning) - collecting and learning how you use smart devices.
  • Remote Control - control smart devices remotely, through an app or website.
  • Voice Control - allows for commands to be spoken to the device.
