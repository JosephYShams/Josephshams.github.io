Welcome to the smart home configuration game.

This game aims to teach you what capabilities smart home devices can perform and the different ways you can interact with the device to limit the privacy risks each capability can introduce.

The rules of the game are as follows:

  • You must purchase all 8 smart home devices.
  • To purchase a device, you must first answer a configuration test. This configuration test evaluates your understanding of smart home device capabilities and what configuration behaviours reduce privacy risks associated with device capabilities.
  • To help you answer the questions, a description of the 5 capabilities introduced in this game is provided on the main page (right-hand side.)
  • There is no time limit for this game, although we expect it to last between 15 and 30 minutes.

Before you start this game, please complete a pre-test (found here).

After completing the pre-test, start audio recording yourself playing this game. Only stop recording when you have completed the game.

On completing the game, please complete a post-test (found here).
